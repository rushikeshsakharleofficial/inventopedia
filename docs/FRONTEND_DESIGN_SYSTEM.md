# Inventopedia Frontend Design System

Inventopedia should feel like a premium editorial SaaS workspace: clean, calm, professional, and mobile-first.

## Direction

Inspired by Notion-style SaaS design, without copying Notion branding, logo, exact layout, or copy.

## Visual Language

- Deep navy hero/dashboard band
- Purple primary CTA
- Warm white canvas
- Soft gray surfaces
- Pastel feature cards
- 12px rounded cards
- 8px rounded buttons
- Thin borders
- Clean database-table visual language
- Product mockup cards
- Simple farm-specific illustrations
